# Python Print

To print to the console use the print method

```
print("Hello")
```

## Print Variables using commas

When printing variables use comma to separate string literal the variables

```
print("Hello", name)
```

Can also have more than one variables

```
print("Hello", name, ", Welcome to ", place)
```

## Print Variables Using Formatted String Literals

If f is placed before string literal than can put variables in {} within the quotes of the string

```
print(f"Hello {name}, Welcome to {place}")
```
