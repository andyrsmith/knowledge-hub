# Copilot-commands

## Context

- @workspace: includes content about your code space. 
- @vscode: ask questions about VS commands
- @terminal: ask questions about terminal commands
- @github: ask about github sklls.  @github What skills are available?

## Variables

- #class ask about a class in a file
- #file add file to add context to question asked.

## Chat modes

- ask: Ask question and get suggestions
- Edit: Can make edits in chosen file, and accept edits on refactoring, debugging , and formatting

Can also response to comments, can generate new workspace, mock data, and guidance on how to implement functiontly like json server


## Slash Commends

- /help help on how to use Copilot Chat
- /optimize Analyzes and improve runtime
- /test create test 
- /doc create documents
- /explain get code explaination
